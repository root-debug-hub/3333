# agar-io-clone
A clone of agar-io implementation with koa2.

目录如下
- middleware    存放自己写的中间件
- models        存放MySQL models
- repository    封装mysql sequelize对数据库的操作
- routes        存放controller

### Usages
```
# build es6 to es2015
$ npm run build

# run the server
# the server listen port 3000 by default
$ npm start
```
